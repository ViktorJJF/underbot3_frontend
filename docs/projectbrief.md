# Project Brief - Underbot3 Frontend

Core requirements and goals for the Underbot3 frontend project.

## Goals
- Build a scalable and maintainable Vue.js frontend for Underbot3.
- Manage betting odds, matches, teams, and user authentication.
- Provide a clean and modern UI using Element Plus and ECharts.

## Scope
- Vue 3 + Vite + TypeScript.
- Vuex for state management.
- Integration with a backend API via Axios.
- Real-time updates via Socket.io.
